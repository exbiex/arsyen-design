# ARSYEN_USER_FLOWS_PRODUCTION.md

# Purpose
Production-grade user flow specification derived from the locked domain model.

# Core Philosophy
- Creative Operating System for Artists
- Work Layer separate from Public Layer
- Create Once, Reference Everywhere
- Global Asset Library
- Universal Visibility
- Universal Versioning
- AI Assists, User Approves

# 1. Onboarding Flow
Landing -> Sign Up -> Verify Email -> Username -> Roles -> Availability -> Profile Photo -> Bio -> Dashboard

# 2. Dashboard Flow
Dashboard acts as a Command Center.
Widgets:
- Tasks Due
- Active Projects
- Notifications
- Conversations
- Recommendations
- Recent Creative Works

# 3. Profile Flow
Profile -> Identity -> Website Builder -> Publish

Profile contains:
- Bio
- Roles
- Availability
- Portfolio
- Resume
- Creative Works
- Custom Pages

# 4. Website Builder Flow
Create Page
Choose Template
Add Sections
Publish

Supported Sections:
- Hero
- Text
- Gallery
- Video
- Featured Works
- Resume
- Timeline
- Contact

# 5. Creative Work Flow
Create Creative Work
Attach Assets

Publish

# 6. Credit Verification Flow
