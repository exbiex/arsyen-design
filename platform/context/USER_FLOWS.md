# USER_FLOWS.md

## Overview
Canonical user flows for Arsyen.

### Core Workflows
- Projects
- Discover
- Feed
- Profile
- Tools

## New User Onboarding
Landing Page → Sign Up → Verify Email → Choose Username → Select Roles → Select Availability → Upload Profile Image → Create Profile → Dashboard

## Dashboard
Login → Dashboard

Displays:
- Tasks Due
- Recent Projects
- Notifications
- Recent Conversations
- Recommendations

## Profile
Edit Profile:
- Bio
- Roles
- Location
- Availability
- Links

Website Builder:
Create Page → Add Sections → Publish

## Creative Works
Create Creative Work:
Title → Description → Work Type → Assets → Credits → Visibility → Publish

Credit Verification:
Notification → Accept / Reject

## Projects
Create Project:
- Blank
- Template
- AI Assisted

Project:
- Action Plans
- Tasks
- Milestones
- Canvases
- Resources
- Files
- Chat

## Canvas
Create Canvas → Add Text / Image / Video / Task / Embed

Canvas Task → Automatically appears in Project Board

## Assets
Upload → Global Asset Library

Reference Asset from:
- Project
- Profile
- Creative Work
- Community

## Communities
Create Community:
- Public
- Private
- Hidden

Join via Discover or Invite

## Discover
Search:
- Artists
- Projects
- Communities

Crew Formation:
- Role
- Availability
- Verified Credits

## Conversations

- Project

- Community
