A square, rounded, dark icon control — dock items, panel-header actions (link, more, share), card add buttons.

```jsx
<IconButton label="Share"><ShareIcon /></IconButton>
<IconButton variant="ghost" active label="Files"><FolderIcon /></IconButton>
<IconButton variant="accent" label="Add"><PlusIcon /></IconButton>
```

Sizes `sm`/`md`/`lg`. `active` tints the icon coral (used for the selected dock item). `variant="accent"` gives a coral-tinted fill.
