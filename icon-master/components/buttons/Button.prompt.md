The coral-red call-to-action and its quieter variants — use for any primary or secondary action.

```jsx
<Button>Message</Button>
<Button variant="secondary">Cancel</Button>
<Button variant="primary" size="lg" fullWidth>Log in</Button>
<Button variant="ghost">Skip</Button>
```

Variants: `primary` (coral fill + glow), `secondary` (dark glass), `ghost` (text-only), `outline`, `danger` (red text on dark). Sizes: `sm` / `md` / `lg`. Pass `icon` / `iconRight` for leading/trailing SVGs, `glow={false}` to drop the halo, `fullWidth` to stretch.
